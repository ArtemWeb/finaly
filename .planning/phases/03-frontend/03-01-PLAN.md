## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → backend (cross-origin in dev) | `next dev` on :3000 calls backend on :8000; untrusted origin crosses here, gated by CORS |
| browser → backend (same-origin in prod) | static export served by FastAPI; same-origin, no CORS |
| build-time env → static bundle | `NEXT_PUBLIC_*` values are inlined into the public client bundle |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-03-01 | Information Disclosure | NEXT_PUBLIC_* env vars | high | mitigate | Only `NEXT_PUBLIC_API_BASE_URL` (non-sensitive base URL) is public; `.env.example` documents nothing secret. `OPENROUTER_API_KEY`/`MASSIVE_API_KEY` stay backend-side only — never prefixed NEXT_PUBLIC_ (V6). |
