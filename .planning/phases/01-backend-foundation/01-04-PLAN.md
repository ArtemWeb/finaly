    Create `backend/app/main.py` per CONVENTIONS.md (module docstring, `from __future__ import annotations`, sorted imports, type hints, module logger).
    Implement async `_snapshot_loop(cache: PriceCache, interval: float) -> None`: loop forever — `await portfolio_service.record_snapshot(cache)` inside a try/except that logs exceptions (never re-raises), then `await asyncio.sleep(interval)`. Read the interval from env `SNAPSHOT_INTERVAL` (default 30.0) in create_app and pass it in so tests can override it.
    Implement async `lifespan(app: FastAPI)` as an `@contextlib.asynccontextmanager`: call `await init_db()`; read `tickers = await get_watchlist_tickers()`; `await app.state.source.start(tickers)`; create the snapshot task via `asyncio.create_task(_snapshot_loop(app.state.cache, app.state.snapshot_interval), name="snapshot-loop")` and store it on app.state; `yield`; on shutdown cancel the snapshot task (await it, suppressing CancelledError) and `await app.state.source.stop()`.
    Implement `create_app() -> FastAPI`: construct `cache = PriceCache()` and `source = create_market_data_source(cache)` (unstarted); create `app = FastAPI(lifespan=lifespan)`; store cache, source, and snapshot_interval on app.state. Register routers BEFORE mounting static: `app.include_router(create_stream_router(cache))`, `app.include_router(create_portfolio_router(cache))`, `app.include_router(create_watchlist_router(cache, source))`; add a `GET /api/health` route returning HTTP 200 with a JSON body like {"status": "ok"} (CORE-02). For static serving (CORE-04): resolve the static directory from env `STATIC_DIR` (default `static`); if that directory exists, `app.mount("/", StaticFiles(directory=static_dir, html=True), name="static")` so the Next.js export and SPA routes are served for all non-API paths; if it does NOT exist (true during Phase 1, before the frontend is built), log a warning and skip the mount so the app still starts cleanly. Return app.
    At module scope add `app = create_app()` so `uvicorn app.main:app --host 0.0.0.0 --port 8000` works.
    Do not register a chat router — the /api/chat route is delivered in Phase 2; structure create_app so a chat router can be added later without restructuring.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser/client → static file serving (non-API routes) | Untrusted request paths resolved against the static export directory |
| process startup → SQLite + background task | Lifecycle code initializes DB and long-running tasks |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-04-01 | Information Disclosure | static file path traversal | high | mitigate | Serve via Starlette StaticFiles(directory=static_dir, html=True), which normalizes paths and confines serving to the directory; no hand-rolled file path joining |
| T-04-02 | Denial of Service | snapshot loop crash kills app or spins hot | medium | mitigate | _snapshot_loop wraps record_snapshot in try/except that logs and continues; fixed await sleep(interval) between iterations |
| T-04-03 | Denial of Service | MarketDataSource fails to stop on shutdown (task/resource leak) | low | mitigate | lifespan cancels the snapshot task and awaits source.stop() on shutdown (idempotent stop per the ABC contract) |
</threat_model>
